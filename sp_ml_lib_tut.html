Signal Processing and Machine Learning Libraries in Python and Tutorials

This section lists some of the commonly used signal processing and machine learning libraries in Python and provides descriptions.

Data Processing and Analysis

  • NumPy
    This is the fundamental package for scientific computing with Python. Useful operations including array manipulation, useful linear algebra, Fourier transform, and random number capabilities.
  • Scipy
    This package includes modules for statistics, optimization, integration, linear algebra, Fourier transforms, signal and image processing, ODE solvers, and more.
  • Pandas
    Powerful tool for data manipulation and handling dataframes.

Machine Learning

  • Sci-kit learn
    The go-to python module for machine learning in Python. Has a wide range of machine learning and optimization algorithms implementations.
  • Keras
    It is a high level API for neural networks and deep learning algorithms. An easy to use starter library for someone who is taking their first steps into deep learning.
  • Tensorflow
    Advanced machine learning platform that provides both high-level and low-level functionalities for implementating custom machine learning and deep learning models. Has a wide range of machine learning algorithm implementations ready to use.

Data Visualization and Plotting

The following libraries offer a wide range of funtionlities for visualizing data such as time series, images and other data types.

  • Matplotlib
  • Seaborn
  • Plotly

Biosignal Processing Libraries

  • MNE
    Open-source Python software for exploring, visualizing, and analyzing human neurophysiological data: MEG, EEG, sEEG, ECoG, and more.
